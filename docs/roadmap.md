# Roadmap

## Phase 1: Foundation

- Vite + TypeScript + Three.js.
- Lint, format, typecheck.
- Vitest and Playwright.
- Basic scene and documents.

## Phase 2: Player Operation

- Movement and camera.
- Attack, dodge, guard, stamina, HP.
- Input abstraction.

## Phase 3: Input Expansion

- Keyboard/mouse.
- Gamepad API.
- Mobile touch UI.
- Debug status.

## Phase 4: Enemy AI

- Grunt, Shield, Fast.
- Chase, attack, hit, death, respawn.

## Phase 5: Map and Exploration

- Compact dungeon.
- Cinder shrines.
- Shortcut door.
- Dropped resource and pickups.
- Boss arena.

## Phase 6: Boss

- Ashen Warden.
- Multiple telegraphed attacks.
- Boss HP bar.
- Defeat state.

## Phase 7: Death and Growth

- Checkpoint respawn.
- Echo drop and recovery.
- Flask refill.
- Simple level-up at shrines.

## Phase 8: Polish

- Fog, lighting, shadows, sound hooks.
- UI improvement.
- Performance pass.
- Browser automation tests.
- Final review, Ready For Review, merge.
