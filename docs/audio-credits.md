# Audio Credits

No third-party audio files are currently included.

Current audio cues are synthesized at runtime through Web Audio in `src/audio/AudioDirector.ts`. Issue #17 did not add file-based audio assets; real BGM/SFX import remains deferred until licensed source files are selected.

When adding third-party audio, record:

- File path.
- Cue name.
- Source URL.
- Author.
- License.
- Retrieval date.
- Notes about edits, looping, or normalization.
